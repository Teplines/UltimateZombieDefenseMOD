# UltimateZombieDefenseMOD
Ultimate Zombie Defense 16 players room MOD
# Update nodes
1. Adjust game values and optimize game experience;
2. After round 50, difficulty rises in waves, and BOSS health is reduced by 50%;
3. Some weapons can automatically empty magazines, and bullet speed is doubled;
4. Player health, money, experience (XP), building limits, and most building health are doubled;
5. Durability of barbed wire, spike wall, chain link fence, military-grade gates, laser fences, TESLA towers, Terminator towers, and other buildings is increased eightfold;
6. Healing stations' health is increased 64 times.
# How to use?
Download the zip file and extract it to the game's root directory.
